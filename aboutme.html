ABOUT ME

I've been learning digital art from junior high school, and how to do game programming and 3D Modelling from the start of my high school year by self-taught. I've made hundreds of digital artwork of my own and has honed my 3D modeling skill from the sculpting, retopology, UV editing and coloring, and Rigging of those models. Outside of university assignments I've been working on my game projects. I've made a short mobile game and a short computer puzzle game where I made all of the assets excluding the text's font.

My main programming language specialities are C# which I mostly use in Unity and Windows Form Application. other than that I also proficient at suing PHP and JavaScript. I have experiences designing and creating database using MySQL and PHPmyadmin for information system at a small clinic. I have experience in web design and programming which include the usage of database. I also have experience in creating mobile application using framework7 and nodejs.
