Customizable Anime Character 3D

A set of 3D models add on package to create character in Unity. The Set provide a custom ui to give user ease in the usage. The user can switch around from cloths, hair and skin color. The set also provide custom anime shader for user to use.

Link to Assets here

Package contain:
A base character with 5 different skin color to choose.
20 different shirt to choose from.
20 different pants and skirt to choose from.
27 jackets and set models including suit and maid outfit.
8 different shoe.
9 front hair, 9 side hair and 21 back hair.
25 hats and accesories to use.
10 eyes, 9 mouth and 7 eyeballs to use
9 basic animation(3 idle animation, 2 walk animation, 2 run animation and jump and landing animation)
Check technical details for full breakdown of vertices.
Polygon count for final model with clothes on will be around 3000 to 4000 vertices and can be lower depending clothes used.
